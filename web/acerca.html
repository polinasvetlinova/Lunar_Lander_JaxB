ACERCA DEL PROYECTO

Autor: Guillermo Cirer
Cliente del proyecto: Mariano
Fecha de creación: Noviembre de 2016
Descripción: El juego consiste en frenar la caída de una nave para que esta pueda aterrizar correctamente.

HERRAMIENTAS DE DESARROLLO

El proyecto ha sido desarrollado mediante código html, css, y javaScript.
El diseño del proyecto tiene dos versiones: una versión responsive, pensada para pantallas de ordenador horizontales, y una versión optimizada para smartphone, con tratamiento vertical.
El juego ha sido diseñado consultando los siguientes enlaces:
  • Para el tratamiento del código html y css: http://www.w3schools.com/html/
  • Para reducir el peso de las imágenes: http://www.compresspng.com/es/
  • Para crear las imágenes .gif: http://www.gifmaker.me/
  • Para editar las imágenes con el programa GIMP: http://www.gimp.org.es/
  • Para controlar y organizar el estado del proyecto mi cuenta en Github: https://www.github.com/guilleCM
  • Además he sacado muchas ideas por parte de mis compañeros de clase (Lluis Cifre y Marc Hernández principalmente)
